# <a name="compiler-warning-level-3-cs0414"></a><span data-ttu-id="78dbb-102">Derleyici Uyarısı (düzey 3) CS0414</span><span class="sxs-lookup"><span data-stu-id="78dbb-102">Compiler Warning (level 3) CS0414</span></span>

<span data-ttu-id="78dbb-104">Bu uyarı, derleyicinin bir değişkene hiçbir şekilde başvurulduğunu doğrulayabileceği çeşitli senaryolarda oluşabilir:</span><span class="sxs-lookup"><span data-stu-id="78dbb-104">This warning can occur in several scenarios in which the compiler can verify that a variable is never referenced:</span></span>

- <span data-ttu-id="78dbb-108">Özel veya iç statik alana yalnızca başlatıcıda sabit bir değer atanır.</span><span class="sxs-lookup"><span data-stu-id="78dbb-108">A private or internal static field is assigned a constant value only in the initializer.</span></span> <span data-ttu-id="78dbb-109">Alanı bir const olarak değiştirmeyi düşünün.</span><span class="sxs-lookup"><span data-stu-id="78dbb-109">Consider changing the field to a const.</span></span>

- <span data-ttu-id="78dbb-110">Özel veya iç alana sabit değerler atanır ve yalnızca #ifdef yönergeleri tarafından dışlanan bloklara kullanılır.</span><span class="sxs-lookup"><span data-stu-id="78dbb-110">A private or internal field is assigned constant values and only used in blocks that are excluded by #ifdef directives.</span></span> <span data-ttu-id="78dbb-111">Alanı #ifdef bloğunun içine yerleştirmeyi göz önünde bulundurun.</span><span class="sxs-lookup"><span data-stu-id="78dbb-111">Consider putting the field inside the #ifdef block.</span></span>

- <span data-ttu-id="78dbb-112">Özel veya iç alana birden fazla konumdaki sabit değerler atanır, ancak başka bir şekilde erişilmez.</span><span class="sxs-lookup"><span data-stu-id="78dbb-112">A private or internal field is assigned constant values in multiple locations but not otherwise accessed.</span></span> <span data-ttu-id="78dbb-113">Alana ihtiyacınız yoksa kaldırmayı göz önünde bulundurun.</span><span class="sxs-lookup"><span data-stu-id="78dbb-113">If you do not need the field, consider removing it.</span></span> <span data-ttu-id="78dbb-114">Aksi takdirde, bunu uygun bir şekilde kullanın.</span><span class="sxs-lookup"><span data-stu-id="78dbb-114">Otherwise, use it in some appropriate way.</span></span>

<span data-ttu-id="78dbb-116">Aşağıdaki örnek, CS0414 üretilecek bir yol gösterir:</span><span class="sxs-lookup"><span data-stu-id="78dbb-116">The following sample shows one way in which CS0414 will be generated:</span></span>

```csharp

{
   private int i = 1;  // CS0414

   public static void Main()
   { }
}
```
