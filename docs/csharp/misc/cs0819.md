## <a name="example-1"></a><span data-ttu-id="5f866-109">Örnek 1</span><span class="sxs-lookup"><span data-stu-id="5f866-109">Example 1</span></span>

```csharp

        var a = 3, b = 2; // CS0819

        // First correction option.
```
